
```bash

# 确认有 index.html 和 assets 文件夹
```

### 第三步：部署到 Vercel
```bash

npm install -g vercel
vercel --prod

# 方法2：GitHub 集成
# 1. 推送代码到 GitHub
# 2. 在 vercel.com 导入项目
# 3. 配置环境变量 GEMINI_API_KEY（如果需要）
```

### 移动端
- 拖拽功能在触摸屏上体验不如桌面端
- 建议主要面向桌面用户使用

### 浏览器兼容性
- 需要支持 ES2020 的现代浏览器
- IE11 不支持（无需考虑）

## 🐛 常见问题

### Q: 用户反馈 API Key 失效？
A: 让用户检查：
1. Key 是否正确复制（有没有多余的空格）
2. Key 是否有足够的额度
3. 网络是否能访问对应的 API 地址

### Q: 构建失败？
A: 检查：
1. `npm install` 是否成功
2. Node.js 版本是否 >= 18
3. 是否有语法错误（运行 `npm run lint`）

## 📞 技术支持

如有问题，可以：
1. 查看浏览器控制台（F12）的错误信息
2. 检查网络请求的响应
3. 清除浏览器缓存重试
